Paint Color Explorer

Created based on the information provided by Dakka Dakka. I didn not find the spreadsheet easy to read, so this page is intended to help painters find alternative colors to common Citadel paints more easily. Color values, HEX values, and equivalent color recommendations are approximations and represent similarity rather than 1-to-1 replacements. Be sure to check any paint itself before using. Not all colors are present/the list is not exhaustive. If you'd like to contribute, please see the Github.
